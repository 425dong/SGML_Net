# SGML_Net: Multimodal Remote Sensing Dataset

This repository contains the preprocessed multimodal remote sensing dataset used in the paper:

> "Semantic-Guided Meta-Learning Network for Multimodal Remote Sensing Classification"
> *Haodong Zhou, C et al.
> Submitted to Remote Sensing (MDPI)

Dataset Contents

- `hsi/` — Preprocessed Hyperspectral Imagery (HSI) patches from MUUFL Gulfport, Houston2013, and Berlin datasets.
- `lidar/` — Corresponding LiDAR elevation/intensity data aligned with the HSI bands.
- `ms/` — Multispectral (MS) imagery (where applicable for the respective scenes).
- `sar/` — Synthetic Aperture Radar (SAR) data (where applicable for the respective scenes).
- `labels/` — Ground truth classification maps (pixel-wise annotations) for all three study sites.
- `README.md` — This file.
- `license.txt` — License information.
